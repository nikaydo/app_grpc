# Пет-проект на gRPC

## Основные эндпоинты

### Веб-интерфейс

- `/login` — вход пользователя  
- `/register` — регистрация  
- `/user` — страница пользователя  

### API

- `/signup` — регистрация через API  
- `/signin` — вход через API
